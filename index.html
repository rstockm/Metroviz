Or load from URL

Your browser will only allow MetroViz to read JSON from another site if that server sends appropriate CORS headers (for example Access-Control-Allow-Origin) that include this app’s origin. Hosting you control (e.g. GitHub Pages, raw files on the same domain, object storage with CORS rules) can work; many generic “paste” or file hosts block cross-origin requests, so the load may fail with a network error. If that happens, download the file and use the dropzone above, or host the JSON where you can set CORS.
